# Bíblia JFA — PWA

## Passo a passo

### 1. Faça upload de todos os arquivos deste ZIP no repositório GitHub

### 2. Faça upload do Biblia_data.js também

### 3. Ative o GitHub Pages
- Vá em Settings → Pages
- Em "Source" selecione: **GitHub Actions**
- Salve

### 4. Aguarde ~2 minutos
O workflow vai rodar automaticamente e publicar o app em:
`https://pruanderley.github.io/NOME-DO-REPOSITORIO/`

### 5. Acesse pelo celular
Abra o link no Chrome → menu (⋮) → "Adicionar à tela inicial"
Vai aparecer o botão **"Instalar"** com o ícone do app!
